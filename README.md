# frontend_servmedic_principal

Angular Principal CORE de MedicDrive.

Este módulo es el núcleo funcional del negocio para especialistas.

Funciones previstas:
- Login especialista
- Dashboard especialista
- Gestión de solicitudes médicas
- Estados de atención
- Ficha médica
- Historial de atenciones
- Estadísticas de clientes y servicios
- Mapa Mapbox
- Pagos demo / placeholder
- Pagos reales en etapa final

Este frontend es el software CORE del negocio.
La app Flutter será soporte móvil de este core.
